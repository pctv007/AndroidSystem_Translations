Qu'est ce qu'Android System :

Android System est l'ultime utilitaire Android combinant plus de 20 outils (Actuellement en développement, plus de fonctions seront ajoutées dans un futur proche)

Nous donnons accès à 2 versions de l'application : Gratuite et Pro.

*** VERSION GRATUITE : ***

Outils Système :
- Explorateur de fichiers MIUI avec la possibilité d'activer l'accès ROOT
- LogCat vous permet d'explorer les logs système

Préférences de l'appli :
- Installeur Framework vous permet d'installer Xposed, requis pour customiser et utiliser les modules. Requiert l'acc�s ROOT et Busybox
- Préférences Application personalisées vous permet de customiser les applis et de choisir les DPI, dispositions, taille de la police, et bien plus. Requiert un framework compatible, l'acc�s ROOT et Busybox

Informations système :
- Benchmarks vous permet de calculer les performances de votre appareil telles que la puissance du CPU, I/O, vitesse de connexion et plus encore 
- Permissions affiche dans une liste détaillée toutes les permissions Android et vous permets de vérifier quelles applications installées utilisent quelles permissions. Celles marquées en rouge sont dangereuses !
- CPU and Kernel spy vous permet d'afficher des informations détaillées à propos du voltage du CPU et d'autres en rapport avec votre Kernel

Editeurs :
- L'éditeur build prop est un outil utile pour éditer manuellement les propriétés du système. Cet outil requiert l'accès ROOT.

Recovery :
- Vous permet de lancer les actions du Recovery directement depuis votre appareil. Ceci nécessite un Recovery, tel que CWM ou TWRP, pour fonctionner correctement

*** VERSION PRO : ***

Sauvegarde :
- La sauvegarde d'application vous permet de sauvegarder et restaurer les données de vos applications. Cet outil requiert l'accès ROOT et Busybox doit êre installé.
- La sauvegarde des données vous permet de sauvegarder et restaurer vos données personnelles telles que les SMS, le journal d'appel, Favoris, Paramètres etc...
- La sauvegarde de propriétés vous permet de sauvegarder et restaurer vos propriétés système. Cet outil nécessite l'accès ROOT

Optimisations Système :
- L'application Zipalign est une tâche qui optimisera toutes vos applications dans les dossiers SYSTEM et DATA. Cet outil requiert l'accès ROOT
- SQL defrag et optimisation est une tâche qui optimise vos bases de données SQL tel que les contacts, etc... Cet outil nécessite l'accès ROOT
- L'application nettoyeur de cache purge le cache des applications et libère de l'espace dans votre SD interne. Cet outil requiert l'accès ROOT

Tweaker :
Combine les tweaks à l'intérieur d'Android Tweaker PRO. Pas encore terminé. Toutes les améliorations seront importées dans cette application aussi vite que possible. Le tweaker nécessite l'accès ROOT pour fonctionner correctement

Réglage du GPS :
- Vous permet de choisir votre pays et d'écraser le fichier de configuration GPS par défaut. Le réglage GPS nécessite l'accès ROOT

Désinstalleur :
- Ce puissant outil vous permet de physiquement retirer les applications système. Cet outil requiert l'accès ROOT

L'application sera plus fournie à chaque mise-à-jour. Nous sommes actuellement en train de travailler pour fournir des traductions pour plus de 30 langages et ajouter encore plus d'utilitaires et d'outils qui feront de cette application le nec plus ultra des utilitaires pour Android !
